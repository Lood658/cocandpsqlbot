# cocandpsqlbot

Это телеграм бот на пайтоне, который может работать с api игры clash of clans и передавать историю поисков кланов в таблицу внутри базы данных postresql

В данном небольшом проекте реализован поиск и получение данных о конкретным клане через тег, можно добавить и другие функций дальше или еще какие то интересные плюшки с базой данных, но я не стал так замарачиваться так как сделал этот проект просто чтобы показать что имею опыт работы с api и базами данных когда буду подавать резюме. Возможно этот проект не сильно связан с моей профессией, но я сделал ее просто чтобы показать что умею что то и имею опыт

Немного о том как все работает:

В файле `databs.py` реализовано подключение, отключение, получение данных и выполнение команд sql 

`set.py` - в этом файле просто технические моменты по типу токена бота и api ключа

`bot.py` - основной файл для запуска бота
